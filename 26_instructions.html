Начало использования

Первый вход в Личный кабинет по ДМС

Перейдите по ссылке dms.psbins.ru или отсканируйте QR код и пройдите регистрацию.

При регистрации укажите:

  • номер мобильного телефона
  • код из СМС, отправленный на указанный номер
  • ФИО полностью
  • номер полиса ДМС
  • дату рождения
  • электронную почту

Укажите свой актуальный номер телефона т.к. дальнейшая авторизация будет происходить через СМС-код.

Рабочий стол

На рабочем столе вы видите:

  1. Данные вашего профиля (ФИО)
  2. Номер страхового полиса ДМС
  3. Срок действия полиса
  4. Телефонные номера для получения медицинской помощи

Добавление полиса ребенка (семейный доступ)

Семейный доступ

У вас есть возможность создать учётную запись и получить доступ к данным только если, вы являетесь законным представителем несовершеннолетнего лица.

Заполните все поля

При регистрации укажите:

  • ФИО
  • Номер страхового полиса
  • Поставьте галочки «Я являюсь представителем ребенка» и «Выражаю свое согласие на обработку персональных данных»
  • Нажмите «Добавить»
  • Подтвердите через СМС
На рабочем столе нажмите «Добавить ребенка»
наверх
Документы застрахованных детей

В разделе «Документы» отображены все данные относящиеся к программе застрахованного несовершеннолетнего.

  1. Полис
  2. Список доступных клиник
  3. Памятка застрахованного лица
  4. Полис путешественника *

Все документы можно сохранить на любом электронном носителе в формате PDF либо распечатать для предъявления при обращении за медицинской помощью.

* Для оформления полиса путешественника (ВЗР) требуется отправить заявку в разделе «Путешественникам» ввести свою Фамилию и Имя на латинице, после чего полис будет доступен для скачивания.

Гарантийные письма
В разделе «Гарантийные письма» отображены ваши «Активные» (действующие) и «Архивные» (срок действия закончился) гарантийные письма
Оформите заявку на получение гарантийного письма через оператора.
наверх
Документы

В разделе «Документы» отображена вся документация, относящиеся к вашей страховой программе:

  • Полис страхования ДМС
  • Список доступных клиник
  • Программа страхования
  • Полис путешественника*

Все документы можно сохранить на любом электронном носителе в формате PDF, либо распечатать для предъявления при обращении за медицинской помощью.

* Для оформления полиса путешественника (ВЗР) требуется отправить заявку в разделе «Путешественникам» ввести свою Фамилию и Имя на латинице, после чего полис будет доступен для скачивания.
наверх
Путешественникам (ВЗР)

В разделе «Путешественникам» отображена вся информация для скачивания и сопровождения вашего полиса выезжающих за рубеж (ВЗР)

  • Оформите заявку в личном кабинете на оформление полиса ВЗР
  • Покрытие полиса по всему Миру исключая РФ
  • При оформлении укажите свою Фамилию и Имя на латинице, как указанно в вашем заграничном паспорте.

После оформления полис будет доступен для скачивания в формате PDF.

наверх
Контакты

В разделе «Контакты» есть вся информация для связи со страховой компанией по вопросам ДМС и ВЗР.

Также можно оставить электронное обращение по следующим вопросам:

  • Клиника не находит меня в списках застрахованных
  • Ошибки в данных Полиса страхования ДМС
  • Хочу застраховать родственника
  • Согласование медицинских услуг
  • Оформление гарантийного письма
  • Ошибки в полисе ВЗР
  • Другой вопрос
